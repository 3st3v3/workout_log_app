# Workout Log GitHub Pages Upload

Upload these files to the root of a GitHub repository.

## Deploy steps
1. Create a new GitHub repository.
2. Upload all files in this folder to the repo root.
3. Make sure `index.html` is in the root, because GitHub Pages looks for `index.html` as an entry file.[web:89][web:90]
4. In the repository, go to **Settings -> Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.[web:93][web:103]
6. Select the `main` branch and the `/root` folder, then save.[web:93][web:94]
7. After deployment, your site URL should follow the pattern `https://<your-username>.github.io/<repository-name>/` for a project site.[web:90][web:94]

## iPhone install
Open the GitHub Pages URL in Safari, tap **Share**, then tap **Add to Home Screen**.[web:69][web:76]
